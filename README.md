# Multi30k(Eng-De)_Dataset_RNN_LSTM
This is my 11th project. This project will be done in a similar style to my 9th project (https://github.com/ris2002/Adult-Census-Income-MLE-NN). I am only going to involve RNN and LSTM, as I am learning those techniques to apply them to regression tasks. For each technique, I will run a total of 30 experiments, i.e., 15 runs each.
## Problem Analysis
* Type-Sequence-to-Sequence Prediction
* Business Goal - The business goal is to identify and translate English sentences into German.
* LSTM/RNN Objective - It needs to get an overall accuracy and F1 score of more than 80% and the precision and recall classes should be similar.
## Dataset Overview
The dataset was exported from Kaggle (https://www.kaggle.com/datasets/hemanthkumar21/multi30k-de-en). It consists of 30,000 rows of English and German sentences; that is the only column.
